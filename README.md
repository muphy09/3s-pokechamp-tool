# 3's PokeChamp Tool

`3's PokeChamp Tool` is an Electron desktop app for Pokemon battle prep and matchup comparison.

## Current Scope

- Online Pokemon search and profile lookup
- Generations 1 through 9
- Alternate and regional forms
- Two 6-slot teams
- `1v1` and `2v2` field layouts
- Hover summaries for stats and weaknesses
- Click-through profile overlays for deeper detail

## Development

```bash
npm install
npm run dev:app
```

For a renderer-only session:

```bash
npm run dev
```

## Packaging

```bash
npm run dist
```
